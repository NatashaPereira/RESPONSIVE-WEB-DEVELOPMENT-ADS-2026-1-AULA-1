Página sobre um animal em extinção desenvolvida para a disciplina de Responsive Web Development, componente do curso de Análise e desenvolvimento de sistemas - UNIVALI.

O código utiliza conceitos básicos de html para a criação de um site sobre o Cuxiú Preto. Escolhi esse animal por ser nativo do Brasil, e uma espécie pouco falada em comparação com a arara azul por exemplo.
Não utilizei conceitos de CSS nesse projeto, apenas o básico do html.
